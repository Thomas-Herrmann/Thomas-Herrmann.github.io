noHandlers :: WebSocketConfig t

No-op handlers for all WebSocket events.

wsOpenSync :: WebSocketData t => URL -> CIO (Maybe (WebSocket t, CIO (Either WebSocketCloseEvent t)))

Open a WebSocket, returning the socket as well as a blocking CIO computation to await incoming data on the socket. When data arrives on the socket, the waiting computation will return Right received_data. When the socket is closed, the waiting computation will return Left close_event.

wsClose :: WebSocket a -> CIO ()

Close a WebSocket. Closing a WebSocket which has been previously closed is a no-op. Any associated wsOnClose handler will not be called.
